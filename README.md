# Ink Counsel Demo

这是李子礽面向游戏法团队实习律师申请场景的个人网站 demo。这里的“实习律师”指取得法律职业资格后的执业实习阶段，不是学生实习生。视觉方向是原创“墨水涂鸦 + 法律作品集”，没有使用 Splatoon 3 的角色、logo、截图或字体素材。

直接在浏览器打开 `index.html` 即可预览。

当前叙事：

- 正式工作经历：两年医疗服务公司一人法务，重点放在隐私合规、服务合同、诉讼支持和业务培训
- 审判辅助经历：中山市第二人民法院民商事案件文书与流程支持
- 游戏法方向：虚拟财产、AI、数据合规、UGC 等专题研究
- 核心表达：不把研究包装成游戏公司实务，而是强调可迁移的合规与合同能力
